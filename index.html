Область разработки

  Начал изучение Frontend с HTML и CSS, изучив основы стал применять полученные навыки на практике и начал разрабатывать свой сайт резюме/портфолио. Вышеупомянутых языков было достаточно для создания страницы такого рода, но я взялся за изучение JavaScript и решил добавить его на свой сайт, сделав обработчики кнопок для портфолио. Захостил зайт на Firebase, приобрел личный домен и привязал его к хостингу, чтобы можно было открыть мой сайт по доменному имени. Начиная разработку сайта сразу стал изучать и Git, чтобы с самого начала вести контроль версий. Проект я опубликовал в свой репозиторий на GitHub!
